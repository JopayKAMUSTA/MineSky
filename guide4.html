What is a Command Block?

A Command Block is a special block in Minecraft that lets you run commands automatically. It's often used in creative mode for automation, maps, or mini-games.

How to Get One

Use this command:

/give @p command_block

Types of Command Blocks

Type Color Description
Impulse Orange Executes once when powered.
Chain Green Runs after the block behind it runs.
Repeat Purple Repeats as long as powered.

Using a Command Block

  1. Place the command block (Creative Mode only).
  2. Right-click to open its menu.
  3. Type a command (e.g. /say Hello!).
  4. Power it with redstone (lever, button, etc.).

Target Selectors

Selector Targets
@p Nearest player
@r Random player
@a All players
@e All entities
@s Itself

Example Commands

  • /tp @p 100 64 100 - Teleport player.
  • /give @p diamond_sword - Give item.
  • /time set day - Set time.
  • /summon lightning_bolt ~ ~ ~ - Summon lightning.

Advanced Uses

  • Create adventure maps.
  • Use with redstone circuits.
  • Chain complex commands.
  • Use scoreboards or data packs.
